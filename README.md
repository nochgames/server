
# NoCH
 NoCH is an independent multiplayer game.
 Technologies used:
 HTML5 Canvas,
 Node JS,
 Matter.js

## server

#### System software requirements

> Node.js 6+


### Quickstart

```bash
git clone https://github.com/nochgames/server.git
cd server
npm install
node server.js
```

## development

```bash
npm install --global nodemon
npm run dev
```
